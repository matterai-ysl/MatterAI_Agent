🔐 SSO测试页面

使用说明：

1. 在A网站登录并获取JWT token

2. 将token粘贴到下面的输入框

3. 点击"测试SSO跳转"按钮

4. 查看控制台日志了解详细过程

🧪 SSO跳转测试

🔍 手动测试URL

你也可以直接访问以下格式的URL进行测试：

本地测试:

http://localhost:9000/auth/sso?token=YOUR_JWT_TOKEN&redirect_to=/

生产环境:

http://47.99.180.80/agent/api/auth/sso?token=YOUR_JWT_TOKEN&redirect_to=/
